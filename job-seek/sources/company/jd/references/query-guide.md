# 京东招聘 — 查询指南

## 数据源信息

- **类型**：公司直招
- **获取方式**：服务端 HTML 解析
- **脚本**：`python scripts/query.py --keyword "关键词"`

## API 详情

- **URL**：`GET https://zhaopin.jd.com/home`
- **解析方式**：正则提取"热招职位"表格中的 `<tr>/<td>` 元素
- 表格列：职位名称、职位类别、工作地点、发布时间

## 返回数据

解析后每行字段：

| 字段 | 说明 |
|------|------|
| 职位名称 | 岗位名称 |
| 职位类别 | 岗位分类 |
| 工作地点 | 工作城市 |
| 发布时间 | 发布日期 |

## 使用示例

```bash
# 基本查询
python scripts/query.py --keyword "AI"
```

## 注意事项

- 无可用 API，依赖 HTML 页面结构解析
- 页面结构变更可能导致解析失败，需维护正则表达式
- 仅返回首页热招职位，数据量有限
